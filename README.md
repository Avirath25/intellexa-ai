INTELLEXA-AI is a high-performance, AI-powered database analysis platform that allows non-technical users to query their databases using plain English.

## ✨ Key Features

- 🔌 **Universal Connectivity**: Support for MySQL, PostgreSQL, and SQLite.
- 💬 **Natural Language Processing**: Advanced logic to handle typos, JOINs, and complex filters.
- 📊 **Dynamic Visualization**: Automatically renders Bar, Line, Pie charts, and KPI cards based on query intent.
- 💡 **AI Insights**: Human-readable data summaries that point out trends and outliers.
- ⚡ **Instant Performance**: Lightning-fast synthesis and rendering within 500ms.
- 📱 **Premium UI**: Modern dark-mode interface with glassmorphism and smooth animations.

## 🚀 Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/intellexa-ai.git
   cd intellexa-ai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Launch the development server**
   ```bash
   npm run dev
   ```

4. **Connect your database**
   - Access the dashboard at `http://localhost:5173`.
   - Select your database type and enter credentials, or use the **Demo Database** to explore immediately.
